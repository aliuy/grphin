About the Data:

We parsed 7,207,357 LinkedIn public profiles (~4GB of raw text) - which resulted in a graph containing 3,215,402 nodes (companies) and 3,574,637 unique edges.

Next we filter out “uninteresting” nodes and edges by setting a threshold. In this case, nodes and edges that appeared in <25 profiles are discarded. The result is a disconnected graph with isolated groups of nodes and edges.

Next, we attempt to build a single connected graph, by running breadth-first-search on an arbitrary node in the original graph. We chose to use IBM as our root node. The resulting graph has 54 companies, along with 102 unique edges.
